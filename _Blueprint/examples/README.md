## 🎯 Mục đích

Examples là **bài mẫu cuối cùng** sau khi đọc các file Blueprint `00 → 07`. Đọc xong examples, bạn nên có thể tự viết bài mới mà không cần hỏi lại.

## 🚀 Khi đọc examples

1. Mở mẫu, đọc qua structure folder
2. Mở từng file, so sánh với template tương ứng trong `../templates/`
3. Quan sát: câu dẫn, hands-on, glossary, navigation footer
4. Khi viết bài thật → copy template + tham chiếu mẫu này

## 📌 Khi nào thêm mẫu mới

- Khi có chủ đề / loại bài đặc biệt mà template không cover hết
