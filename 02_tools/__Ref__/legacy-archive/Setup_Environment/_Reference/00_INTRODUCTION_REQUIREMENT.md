## 📚 Thuật ngữ

| Thuật ngữ | Tiếng Anh | Nghĩa |
|-----------|-----------|-------|
| DevOps | Development + Operations | Phát triển + Vận hành |
| CI | Continuous Integration | Tích hợp liên tục |
| CD | Continuous Delivery/Deployment | Triển khai liên tục |
| Agile | Agile Methodology | Phương pháp linh hoạt |
| Scrum | Scrum Framework | Khung làm việc Scrum |
| Sprint | Sprint | Chu kỳ phát triển ngắn |
| Standup | Daily Standup | Họp nhanh hàng ngày |
| Backlog | Product Backlog | Danh sách công việc |
| IaC | Infrastructure as Code | Hạ tầng dưới dạng code |
| SRE | Site Reliability Engineering | Kỹ sư độ tin cậy |
