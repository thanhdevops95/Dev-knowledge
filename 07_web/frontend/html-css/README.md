## 📖 Lessons — Cụm Basic (5 bài)

Năm bài đi theo trình tự dựng một trang web hoàn chỉnh: hiểu vai trò HTML/CSS, viết cấu trúc semantic, làm form truy cập được, nắm cơ chế style, rồi dựng bố cục responsive.

| # | Bài | Trọng tâm | Tag |
|---|---|---|---|
| 00 | [HTML & CSS là gì?](lessons/01_basic/00_what-is-html-and-css.md) | Vai trò HTML (cấu trúc) vs CSS (trình bày), cách trình duyệt render | MUST-KNOW |
| 01 | [HTML Essentials](lessons/01_basic/01_html-essentials.md) | Semantic tags, cấu trúc tài liệu, các phần tử thường dùng | MUST-KNOW |
| 02 | [Forms & Accessibility](lessons/01_basic/02_forms-and-accessibility.md) | Input đúng cách, label, ARIA cơ bản, a11y | MUST-KNOW |
| 03 | [CSS Fundamentals](lessons/01_basic/03_css-fundamentals.md) | Selectors, Specificity, Box Model | MUST-KNOW |
| 04 | [Layout](lessons/01_basic/04_layout-flexbox-grid-responsive.md) | Flexbox, Grid, Responsive Design | MUST-KNOW |
