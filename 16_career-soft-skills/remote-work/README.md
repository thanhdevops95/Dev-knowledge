---

## 🎯 Mục tiêu tổng

Sau khi đi qua chủ đề này, bạn sẽ:
- [x] Hiểu mô hình **onsite/hybrid/remote/distributed** + vì sao **async-first** là xương sống
- [x] Dựng **home office** + bộ công cụ remote, đặt ranh giới work/life
- [x] Cộng tác **async xuyên múi giờ** (overlap hours, handoff, documentation default)
- [x] Giữ **năng suất + deep work** tại nhà, đo bằng **output** thay vì giờ online, chống always-on
- [x] Chăm sóc **wellbeing** (chống cô đơn/burnout) + xây **niềm tin & văn hoá đội remote**
