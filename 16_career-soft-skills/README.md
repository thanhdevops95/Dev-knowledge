## 🎯 Chủ đề này có gì

Career path · Interview prep · Communication · Technical writing · Agile/Scrum · Learning how to learn · Remote work · Time management.

## 📂 Các chủ đề con (L2) — Basic hoàn chỉnh ✅

| Cụm | Trạng thái | Nội dung |
|---|---|---|
| [career-path](./career-path/) | ✅ 5/5 | Bản đồ vai trò + nấc thang, roadmap kỹ năng, CV/portfolio, tìm việc/offer, thăng tiến. |
| [interview-prep](./interview-prep/) | ✅ 5/5 | Quy trình phỏng vấn, coding/DSA, system design, behavioral/STAR, kế hoạch ôn + mock. |
| [communication](./communication/) | ✅ 5/5 | Async/viết, họp/standup, feedback (SBI) + code review, stakeholder non-tech. |
| [technical-writing](./technical-writing/) | ✅ 5/5 | README/docs, design doc/RFC + ADR, API/code docs, sơ đồ (mermaid/C4). |
| [agile-scrum](./agile-scrum/) | ✅ 5/5 | Agile mindset, Scrum, Kanban, user stories/estimation, thực chiến/pitfalls. |
| [learning-how-to-learn](./learning-how-to-learn/) | ✅ 5/5 | Cơ chế học, kỹ thuật (active recall/spaced rep), deliberate practice, ghi chú, thói quen. |
| [remote-work](./remote-work/) | ✅ 5/5 | Mô hình remote, setup, async collab, năng suất, wellbeing/văn hoá đội. |
| [time-management](./time-management/) | ✅ 5/5 | Ưu tiên, deep work/time blocking, GTD/planning, nhịp độ bền vững. |

> **Tổng: 8 cụm × 5 bài = 40 bài Basic.** (Intermediate dự kiến ở Phase 2.)
