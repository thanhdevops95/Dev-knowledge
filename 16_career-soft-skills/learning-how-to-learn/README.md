# 🧠 Learning How to Learn — Học cách học cho dev

> **Tác giả:** Mr.Rom\
> **Phiên bản:** v1.0.0\
> **Tạo lúc:** 20/05/2026\
> **Cập nhật:** 15/06/2026

> 🎯 *Meta-skill nền tảng nhất với dev: công nghệ đổi liên tục nên "biết cách học" quan trọng hơn biết một công nghệ cụ thể. Học cơ chế não học, kỹ thuật evidence-based (active recall, spaced repetition), luyện tập có chủ đích + học qua dự án, quản lý thông tin/ghi chú, và xây thói quen học bền tránh burnout.*

---

## 🎯 Mục tiêu tổng

Sau khi đi qua chủ đề này, bạn sẽ:
- [x] Hiểu **cơ chế học** (working/long-term memory, cognitive load, focused vs diffuse)
- [x] Áp dụng kỹ thuật hiệu quả: **active recall, spaced repetition, interleaving, Feynman** — tránh illusion of competence
- [x] **Deliberate practice** + học qua **dự án**, dùng debugging/feedback làm cơ hội học
- [x] Quản lý thông tin: ghi chú (Zettelkasten/PARA nhẹ), second brain, learning in public
- [x] Xây **thói quen học bền** (Atomic Habits, consistency > intensity), growth mindset, tránh tutorial hell + burnout
