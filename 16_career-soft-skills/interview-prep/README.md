## 🎯 Mục tiêu tổng

Sau khi đi qua chủ đề này, bạn sẽ:
- [x] Nắm **quy trình phỏng vấn** tech end-to-end + khác biệt theo loại công ty
- [x] Có **khung tiếp cận coding interview** + ôn DSA cốt lõi theo pattern, biết Big-O
- [x] Trả lời **system design interview** bằng framework 7 bước + building blocks
- [x] Kể chuyện **behavioral** thuyết phục bằng **STAR**, chuẩn bị kho story
- [x] Lập **kế hoạch ôn 4-8 tuần** + luyện **mock interview**, xử lý lo lắng, học từ rejection
