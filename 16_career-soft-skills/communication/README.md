# 💬 Communication — Giao tiếp hiệu quả cho dev

> **Tác giả:** Mr.Rom\
> **Phiên bản:** v1.0.0\
> **Tạo lúc:** 20/05/2026\
> **Cập nhật:** 14/06/2026

> 🎯 *Kỹ năng giao tiếp ở nơi làm việc — yếu tố thường quyết định thăng tiến hơn cả kỹ thuật ở mức senior+. Học giao tiếp async/viết, họp + trực tiếp, cho/nhận feedback + xử lý bất đồng, và giao tiếp với stakeholder non-tech. Nhiều template + checklist + ví dụ before/after.*

---

## 🎯 Mục tiêu tổng

Sau khi đi qua chủ đề này, bạn sẽ:
- [x] Chọn đúng **kênh + đối tượng** giao tiếp (sync/async, dev/PM/sếp/non-tech)
- [x] Viết **async** rõ ràng (BLUF, no-hello, bug report, documentation mindset)
- [x] Chạy **họp/standup** hiệu quả + lắng nghe chủ động + trình bày mạch lạc
- [x] Cho/nhận **feedback** (SBI), giao tiếp **code review** không tổn thương, xử lý bất đồng (disagree & commit)
- [x] Giao tiếp với **stakeholder non-tech**: quản lý kỳ vọng, nói "không", báo tin xấu sớm
