## 🎯 Learning Objectives (Mục tiêu học tập)

After this module, you will (Sau module này, bạn sẽ):

- ✅ Understand the function of key DevOps tools (Git, Docker, VS Code, etc.) *(Hiểu chức năng các công cụ DevOps chính)*
- ✅ Install and configure a complete DevOps environment on your OS *(Cài đặt và cấu hình môi trường DevOps hoàn chỉnh trên máy bạn)*
- ✅ Verify the installation to ensure readiness for learning *(Kiểm tra cài đặt để đảm bảo sẵn sàng cho việc học)*

This module is divided into two parts: **Theory** (Understanding tools) and **Practice** (Installation).
*(Module này chia làm 2 phần: **Lý thuyết** (Hiểu công cụ) và **Thực hành** (Cài đặt).)*

### 1. Theory: Tools Overview
Before installing, you should understand WHAT you are installing and WHY.
