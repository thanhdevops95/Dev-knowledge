## Giới thiệu

### Go là gì?

Go (Golang) là ngôn ngữ lập trình được phát triển bởi Google năm 2009.

**Đặc điểm:**
- **Fast**: Compiled language, performance cao
- **Simple**: Cú pháp đơn giản, dễ học
- **Concurrent**: Built-in concurrency với goroutines
- **Static typing**: Type-safe

### Tại sao DevOps cần Go?

1. **Performance**: Nhanh hơn Python, Node.js
2. **DevOps Tools**: Docker, Kubernetes, Terraform viết bằng Go
3. **CLI Tools**: Dễ build CLI applications
4. **Microservices**: Phù hợp cho microservices
