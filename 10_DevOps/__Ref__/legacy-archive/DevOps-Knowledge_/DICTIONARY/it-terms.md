## A

### Algorithm -- Thuật toán
- **Definition -- Định nghĩa:** A set of logical steps to solve a problem. -- Tập hợp các bước logic để giải quyết một vấn đề.
- **Example -- Ví dụ:** Sorting algorithm, Search algorithm -- Thuật toán sắp xếp, Thuật toán tìm kiếm

### API (Application Programming Interface -- Giao diện Lập trình Ứng dụng)
- **Definition -- Định nghĩa:** A set of rules and protocols that allow applications to communicate with each other. -- Tập hợp các quy tắc và giao thức cho phép các ứng dụng giao tiếp với nhau.
- **Types -- Loại:** REST API, GraphQL API, SOAP API

### Architecture -- Kiến trúc
- **Definition -- Định nghĩa:** The overall design and structure of a system. -- Thiết kế và cấu trúc tổng thể của một hệ thống.
- **Types -- Loại:** Monolithic, Microservices, Serverless

### Authentication -- Xác thực
- **Definition -- Định nghĩa:** The process of verifying a user's identity. -- Quá trình xác minh danh tính người dùng.
- **Methods -- Phương thức:** Password, MFA, OAuth, JWT

### Authorization -- Phân quyền
- **Definition -- Định nghĩa:** The process of determining a user's access rights. -- Quá trình xác định quyền truy cập của người dùng.
- **Types -- Loại:** RBAC, ABAC, ACL

## B

### Backend -- Phần Server
- **Definition -- Định nghĩa:** The server-side of an application that handles logic and data. -- Phần phía server của ứng dụng xử lý logic và dữ liệu.
- **Components -- Thành phần:** Server, Database, API

### Backup -- Sao lưu
- **Definition -- Định nghĩa:** Creating copies of data for recovery purposes. -- Tạo bản sao dữ liệu cho mục đích phục hồi.
- **Types -- Loại:**
  - Full backup -- Sao lưu đầy đủ
  - Incremental backup -- Sao lưu tăng dần
  - Differential backup -- Sao lưu khác biệt

### Bandwidth -- Băng thông
- **Definition -- Định nghĩa:** The maximum amount of data that can be transmitted over a network in a given time. -- Lượng dữ liệu tối đa có thể truyền qua mạng trong một khoảng thời gian.
- **Unit -- Đơn vị:** Mbps, Gbps

### Binary -- Nhị phân
- **Definition -- Định nghĩa:** A number system using only 0 and 1. -- Hệ thống số chỉ sử dụng 0 và 1.
- **Use -- Sử dụng:** Machine code, File representation -- Mã máy, Biểu diễn file

### Bug -- Lỗi
- **Definition -- Định nghĩa:** An error or flaw in software that causes unexpected behavior. -- Lỗi hoặc khuyết điểm trong phần mềm gây ra hành vi không mong muốn.
- **Types -- Loại:** Syntax error, Logic error, Runtime error -- Lỗi cú pháp, Lỗi logic, Lỗi runtime

## C

### Cache -- Bộ nhớ đệm
- **Definition -- Định nghĩa:** Temporary storage for faster data access. -- Lưu trữ tạm thời để truy cập dữ liệu nhanh hơn.
- **Types -- Loại:**
  - Browser cache -- Cache trình duyệt
  - Application cache -- Cache ứng dụng
  - CDN cache -- Cache CDN

### Client -- Máy khách
- **Definition -- Định nghĩa:** A device or application that requests services from a server. -- Thiết bị hoặc ứng dụng yêu cầu dịch vụ từ server.
- **Example -- Ví dụ:** Web browser, Mobile app -- Trình duyệt web, Ứng dụng di động

### Cloud Computing -- Điện toán Đám mây
- **Definition -- Định nghĩa:** Delivery of IT resources over the internet. -- Cung cấp tài nguyên IT qua internet.
- **Models -- Mô hình:** IaaS, PaaS, SaaS

### Compiler -- Trình Biên dịch
- **Definition -- Định nghĩa:** Translates source code into machine code. -- Chuyển đổi mã nguồn thành mã máy.
- **Example -- Ví dụ:** GCC, Clang, javac

### Concurrency -- Đồng thời
- **Definition -- Định nghĩa:** Multiple tasks executing in overlapping time periods. -- Nhiều tác vụ thực thi trong các khoảng thời gian chồng chéo.
- **Related -- Liên quan:** Parallelism, Threading

## D

### Database -- Cơ sở Dữ liệu
- **Definition -- Định nghĩa:** An organized collection of structured data. -- Tập hợp dữ liệu có cấu trúc được tổ chức.
- **Types -- Loại:**
  - SQL: MySQL, PostgreSQL, SQL Server
  - NoSQL: MongoDB, Redis, Cassandra

### Debugging -- Gỡ lỗi
- **Definition -- Định nghĩa:** The process of finding and fixing bugs in code. -- Quá trình tìm và sửa lỗi trong code.
- **Tools -- Công cụ:** Debugger, Logging, Print statements

### Dependency -- Phụ thuộc
- **Definition -- Định nghĩa:** External code or library that a project relies on. -- Code hoặc thư viện bên ngoài mà dự án phụ thuộc vào.
- **Management -- Quản lý:** npm, pip, go mod, Maven

### Deployment -- Triển khai
- **Definition -- Định nghĩa:** The process of releasing software to a production environment. -- Quá trình phát hành phần mềm lên môi trường production.
- **Types -- Loại:** Manual, Automated, Continuous

### DNS (Domain Name System -- Hệ thống Tên miền)
- **Definition -- Định nghĩa:** Translates domain names to IP addresses. -- Chuyển đổi tên miền thành địa chỉ IP.
- **Records -- Bản ghi:** A, AAAA, CNAME, MX, TXT

## E

### Encryption -- Mã hóa
- **Definition -- Định nghĩa:** Converting data into an unreadable format for security. -- Chuyển đổi dữ liệu thành định dạng không đọc được để bảo mật.
- **Types -- Loại:**
  - Symmetric: AES, DES -- Đối xứng
  - Asymmetric: RSA, ECC -- Bất đối xứng

### Endpoint -- Điểm cuối
- **Definition -- Định nghĩa:** A specific URL in an API to access a resource. -- URL cụ thể trong API để truy cập tài nguyên.
- **Example -- Ví dụ:** `GET /api/users`, `POST /api/orders`

### Environment -- Môi trường
- **Definition -- Định nghĩa:** A configured setting where software runs. -- Cài đặt được cấu hình nơi phần mềm chạy.
- **Types -- Loại:** Development, Staging, Production -- Phát triển, Staging, Production

### Environment Variable -- Biến Môi trường
- **Definition -- Định nghĩa:** System-level variables used to configure applications. -- Biến cấp hệ thống dùng để cấu hình ứng dụng.
- **Example -- Ví dụ:** `PATH`, `HOME`, `DATABASE_URL`

## F-L

### Framework -- Khung công việc
- **Definition -- Định nghĩa:** A pre-built structure for developing applications. -- Cấu trúc được xây dựng sẵn để phát triển ứng dụng.
- **Example -- Ví dụ:** React, Django, Spring Boot

### Frontend -- Phần Giao diện
- **Definition -- Định nghĩa:** The client-side of an application that users interact with. -- Phần phía client của ứng dụng mà người dùng tương tác.
- **Technologies -- Công nghệ:** HTML, CSS, JavaScript

### Gateway -- Cổng
- **Definition -- Định nghĩa:** An entry point that routes requests in a network or API. -- Điểm vào định tuyến các requests trong mạng hoặc API.
- **Types -- Loại:** API Gateway, Payment Gateway

### Latency -- Độ trễ
- **Definition -- Định nghĩa:** The time delay between a request and response. -- Độ trễ thời gian giữa request và response.
- **Measurement -- Đo lường:** Milliseconds (ms)

### Library -- Thư viện
- **Definition -- Định nghĩa:** Reusable code that provides specific functionality. -- Code có thể tái sử dụng cung cấp chức năng cụ thể.
- **Example -- Ví dụ:** lodash, requests, axios

## M-R

### Middleware -- Phần mềm Trung gian
- **Definition -- Định nghĩa:** Software that acts as a bridge between components. -- Phần mềm đóng vai trò cầu nối giữa các thành phần.
- **Example -- Ví dụ:** Authentication middleware, Logging middleware

### Proxy -- Proxy
- **Definition -- Định nghĩa:** An intermediary server between client and target server. -- Server trung gian giữa client và server đích.
- **Types -- Loại:**
  - Forward proxy -- Proxy chuyển tiếp
  - Reverse proxy -- Proxy đảo ngược

### Queue -- Hàng đợi
- **Definition -- Định nghĩa:** A data structure following FIFO (First In, First Out). -- Cấu trúc dữ liệu theo FIFO (Vào trước, Ra trước).
- **Use -- Sử dụng:** Message queues, Task processing -- Hàng đợi tin nhắn, Xử lý tác vụ

### Repository -- Kho lưu trữ
- **Definition -- Định nghĩa:** A storage location for code and version history. -- Vị trí lưu trữ code và lịch sử phiên bản.
- **Platforms -- Nền tảng:** GitHub, GitLab, Bitbucket

### Runtime -- Thời gian Chạy
- **Definition -- Định nghĩa:** The environment where code executes. -- Môi trường nơi code thực thi.
- **Example -- Ví dụ:** Node.js runtime, Python interpreter

## S-Z

### Server -- Máy chủ
- **Definition -- Định nghĩa:** A computer that provides services to clients. -- Máy tính cung cấp dịch vụ cho clients.
- **Types -- Loại:** Web server, Database server, Application server

### Stack -- Ngăn xếp / Stack công nghệ
- **Definition -- Định nghĩa:** A set of technologies used together, or a LIFO data structure. -- Tập hợp công nghệ được sử dụng cùng nhau, hoặc cấu trúc dữ liệu LIFO.
- **Example -- Ví dụ:** MEAN stack, LAMP stack

### Thread -- Luồng
- **Definition -- Định nghĩa:** A unit of execution within a process. -- Đơn vị thực thi trong một tiến trình.
- **Related -- Liên quan:** Multi-threading, Concurrency

### Token -- Token
- **Definition -- Định nghĩa:** A piece of data used for authentication or authorization. -- Một phần dữ liệu dùng để xác thực hoặc phân quyền.
- **Types -- Loại:** Access token, Refresh token, JWT

### Version Control -- Quản lý Phiên bản
- **Definition -- Định nghĩa:** A system that tracks changes to code over time. -- Hệ thống theo dõi các thay đổi của code theo thời gian.
- **Tools -- Công cụ:** Git, SVN, Mercurial

### Webhook -- Webhook
- **Definition -- Định nghĩa:** An HTTP callback triggered by an event. -- HTTP callback được kích hoạt bởi một sự kiện.
- **Use -- Sử dụng:** Real-time notifications, Integrations -- Thông báo real-time, Tích hợp
