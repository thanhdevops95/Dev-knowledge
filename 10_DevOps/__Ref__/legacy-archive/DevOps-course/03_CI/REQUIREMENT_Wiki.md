# 📋 MODULE 03: CI - YÊU CẦU & TIÊU CHÍ NGHIỆM THU

## 🎯 Mục tiêu Module

1. ✅ Hiểu Continuous Integration (CI) và lợi ích
2. ✅ Setup GitHub Actions workflows
3. ✅ Viết automated tests (unit, integration)
4. ✅ Implement code quality checks (linting, formatting)
5. ✅ Tự động build và push Docker images

## 📖 Thuật ngữ

| Từ viết tắt | Tiếng Anh đầy đủ | Nghĩa tiếng Việt |
|-------------|------------------|------------------|
| **CI** | Continuous Integration | Tích hợp liên tục |
| **Pipeline** | CI/CD Pipeline | Quy trình tự động hóa |
| **Workflow** | GitHub Actions Workflow | Luồng công việc tự động |
| **Job** | Job | Công việc trong workflow |
| **Step** | Step | Bước trong job |
| **Artifact** | Build Artifact | Sản phẩm build ra |
| **Runner** | GitHub Runner | Máy chạy workflow |
