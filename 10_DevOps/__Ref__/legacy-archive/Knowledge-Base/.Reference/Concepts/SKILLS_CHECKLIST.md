# ✅ DevOps Skills Checklist

> **Đánh dấu khi bạn đã master mỗi skill**

---

## 📌 Cách sử dụng

1. Fork repo này về account cá nhân
2. Chỉnh sửa file này khi học xong mỗi phần
3. Thay `[ ]` thành `[x]` khi hoàn thành
4. Push lên GitHub để track tiến độ
