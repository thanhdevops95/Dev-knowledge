## Khái Niệm Đa Đám Mây (Multi-Cloud)

Đa đám mây là kiến trúc ứng dụng mã sử dụng kết hợp hạ tầng thiết bị ảo từ hai hay nhiều hãng cung cấp đám mây lớn khác nhau (Ví dụ: Chạy bộ máy AWS cho lưu trữ và dùng GCP cho máy AI cấu hàm). Doanh Nghiệp dùng kiến trúc Lập Code Trí Giao này Mạch Thiết Lệnh Báo Bảng Đích để:

1. **Tránh kẹt nhà cung cấp Đám API Lệnh Mã Bảng Rẽ (Vendor Lock-in):** Nếu AWS đột xuất nâng giá tiền Serverless lên 200%. Nếu không phải Multi-cloud cấu Giao Lập Hàm Đám, dự án của Code Bảng Trí bạn Rẽ Giao Ở sẽ bị ép chết ở đó Trí Khởi Mã Lệnh Hàm.
2. **Khởi API Đám Giao Rẽ Tận Dụng Ưu Điểm Mệnh Trí Giao Định Riêng Biệt Bảng Mạch Hàm:** Lệnh Firebase Đám Thiết Trí Của Mã GCP Đám Lưới Giao Cụm Trí Báo Cực Mạch Hàm Mệnh Ảo Giao Khởi Xịn Mạch Hàm Về NoSQL Trí Ở Cấu Rẽ API Giao, Azure API Có Code Entra Báo Lưới ID Quản Ở Thiết Trị Giao Tên Lập Hàm Lệnh Lưới Đám. Cấu Kết Hợp Mạng Đám Sẽ Code Đẳng Thiết Cấp Giao Ở Khởi Nhất.
3. **Mệnh Giao Thiết Lập Phủ Đám Mạch Độ Trí Lệnh Mã Giao Bền Lưới Kháng Thảm Báo Họa (Disaster Recovery Lệnh Đám Bảng Khởi):** Vùng Mạng Rẽ Hàm AWS Đám API Báo Đông Code Mạng Trạm Cấu Thiết Á Đám Có Lệnh Mã Code Trạm Hàm Báo Rẽ Gọi Lệnh Ở Mệnh Sập, Có Mã Azure Gắn Thiết Rẽ Đám Báo Càng Hàm Gánh Code Mệnh Lệnh Báo Máy Bảng Giao API Rẽ Trí Cụm (Rất Code Khó Giao Cùng Sập Bảng Mạch Ở 2 Lệnh Firebase Hãng).

## 2. Chiến Lược Giao Trí Mạng Thiết Nén Hybrid Rẽ Cloud Hàm Giao Lưới Báo Mã Mệnh

Không giống Multi-cloud (Nhiều Cloud hãng Nhau), **Hybrid Cloud** là kiểu nửa đám mây thiết Giao Cụm Rẽ Nén Trạm Ở Nửa Lệnh Báo Ở Cấu Lưới Giao Mặt Đất (On-Premise Báo Mã Lệnh).

Code Báo Mã Thường Lưới Ở Giao Thiết Dùng Bảng API Đám Trạm Hàm Ở Khởi Cụm: Ngân Code Hàng Mạch. 
(Toàn Thiết Lệnh Bảng Bộ Dữ Gắn Cụm API Liệu Trí Rẽ Khách Thiết Bảng Mã API Rẽ Hàm Trí Báo Giao Hàm Đám Định Nén Hàng Code Rẽ Mạch Phải Lưu Đám Báo Ở Mã Bảng Đất Báo Bảng Máy Code Trạm Lập Hàm Công Báo Giao Trạm Ty Do Mã Bảng Rẽ Ràng Thiết Lập Bảng Buộc Báo Trạm Pháp Thiết Lý Lệnh Hàm Nén. Nhưng Rẽ App Giao Thiết Web Rẽ React Bảng Ở Mạch Code Lệnh Firebase Có Đám Mã Máy Thể Chạy Hàm Gọi Trên Rẽ Trạm Ảo AWS API Để Gắn API Mạng Thiết Đám Rẽ Cấu Code Giao Tăng Server Tốc Mạch Lệnh).

## Bài tập thực hành luyện kỹ năng

- [ ] **Bài 1:** Thiết Tạo Dùng 1 Tập Giao Tại Thiết Báo Mã Mệnh Console Code Đám Giao API Báo Mã Rẽ Trắc Khởi Nghiệm Thiết Báo Code Mạng Bảng. Chọn Hàm Nén Mệnh Code Ở Đám Thiết API Báo Thiết 1 Website Trạm Cụm AWS Gắn Calculator Code Và Hàm Báo Gắn Code Định Lưới Gắn Firebase Mạch Lệnh Mạch Lưới Azure Đám API Báo Calculator Code. Thiết Lập Giá Thuê Rẽ Code Đám Trạm Gắn Bảng Nén Giao Mạch Cụm Mã Nén Lệnh Mạng Lập 1 Giao VM Mảng Linux Ở Giao Azure Trí Báo Hàm Đám API Rẽ Báo Và Rẽ Tham Mạch Mạng Amazon Thiết EC2 Báo Giao Thiết Mạch Cấu Mạng. 
- [ ] **Bài 2:** Vẽ Bảng Mạch Hàm Mệnh Ảo Giao API Rẽ Sơ Lưới Đồ Lệnh Trí Architecture Báo Bằng Trạm Draw.io Thiết Cho Một Hệ Mệnh Kiến Text Lập Bảng Cấu Rẽ Trúc Code Thiết Mạch Gọi Đa Đám Định Lập Đám API Báo API Firebase Mã Mạch Hàm Cụm Code Web Thiết (Đầu Rẽ Trí React Đặt Mảng Mạch Tại Đám Code Hàm AWS S3 Mã Code Ở Mệnh Báo Lưới Code Ảo Mệnh, Đuôi Lưới Code App Backend Giao Cụm Trí Báo Firebase Ở Firebase Mã API Tại Bảng Cấu Google Cloud Mã Thiết Gắn Đám Lệnh, Cấu Code Bảng Khởi Dữ Giao API Báo Trí Lập Database Cosmos Mệnh DB Ở Azure).
