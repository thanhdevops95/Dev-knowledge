## 🎯 Chủ đề này có gì

Phát triển ứng dụng *mobile*: iOS với Swift, Android với Kotlin, các framework *cross-platform* (React Native, Flutter), và kiến trúc ứng dụng di động. Mỗi cụm đã có lộ trình **Basic 5 bài** hoàn chỉnh — code chạy được, có sơ đồ + cheatsheet + glossary.
