# 🍎 iOS (Swift + SwiftUI) — Lập trình iOS native

> **Tác giả:** Mr.Rom\
> **Phiên bản:** v1.0.0\
> **Tạo lúc:** 16/05/2026\
> **Cập nhật:** 15/06/2026

> 🎯 *Lập trình iOS native bằng **Swift** + **SwiftUI** (declarative, khuyến nghị 2026). Học Swift cơ bản (optionals, struct/class, protocol), SwiftUI (view/modifier/@State), data + state (@Observable, networking, SwiftData) + navigation, và build/publish qua Xcode → TestFlight → App Store. Yêu cầu máy Mac + Xcode. CÓ code Swift chạy được (Swift 6, iOS 17+).*

---

## 🎯 Mục tiêu tổng

Sau khi đi qua chủ đề này, bạn sẽ:
- [x] Hiểu hệ sinh thái iOS dev (Swift, Xcode, UIKit vs **SwiftUI**) + yêu cầu Mac/Apple Developer
- [x] Viết **Swift** đủ dùng: optionals, struct vs class, protocol, closure, async/await
- [x] Dựng UI bằng **SwiftUI** (view/modifier/@State/@Binding) + preview
- [x] Quản lý data/state với **@Observable** + navigation + networking (URLSession/Codable) + **SwiftData**
- [x] Build + **TestFlight/App Store** (signing, provisioning, review của Apple)
