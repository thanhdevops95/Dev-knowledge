## рҹҗҚ**CбәӨU TRГҡC Dб»° ГҒN PYTHON**

### Dб»ұ ГЎn Д‘ЖЎn giбәЈn (Script)

```
my-script/
в”ңв”Җв”Җ main.py               # File chГӯnh
в”ңв”Җв”Җ README.md             # HЖ°б»ӣng dбә«n
в”ңв”Җв”Җ requirements.txt      # Dependencies
в”ңв”Җв”Җ .gitignore           # Ignore git
в””в”Җв”Җ venv/                # Virtual environment (khГҙng commit)
```

### Dб»ұ ГЎn vб»«a (Application)

```

в”Ӯ       в””в”Җв”Җ module2.py

в”Ӯ   в””в”Җв”Җ test_utils.py

в”ңв”Җв”Җ logs/                # Log files (khГҙng commit)
в”ңв”Җв”Җ docs/                # TГ i liб»Үu
```

## рҹ“Ғ**CГҒC THЖҜ Mб»ӨC CHUNG**

| ThЖ° mб»Ҙc | Mб»Ҙc Д‘Гӯch |
|---------|----------|
| `src/` | Source code chГӯnh |
| `tests/` hoбә·c `test/` | Unit tests |
| `docs/` | TГ i liб»Үu |
| `scripts/` | Scripts hб»— trб»Ј (build, deploy) |
| `config/` | File cбәҘu hГ¬nh |
| `data/` | Dб»Ҝ liб»Үu mбә«u, fixtures |
| `logs/` | Log files |
| `public/` | Static files (web) |
| `assets/` | Images, fonts |
| `examples/` | VГӯ dб»Ҙ sб»ӯ dб»Ҙng |
| `.github/` | GitHub Actions, templates |

## вң…**QUY Tбә®C Дҗбә¶T TГҠN**

### ThЖ° mб»Ҙc

| Convention | VГӯ dб»Ҙ | DГ№ng cho |
|------------|-------|----------|
| lowercase | `models`, `utils` | Phб»• biбәҝn nhбәҘt |
| snake_case | `my_module` | Python |
| kebab-case | `my-component` | Node.js, web |
| PascalCase | `MyClass` | C#, Java (cho class) |

### File

| NgГҙn ngб»Ҝ | Convention | VГӯ dб»Ҙ |
|----------|------------|-------|
| Python | snake_case | `my_module.py` |
| JavaScript | camelCase hoбә·c kebab-case | `myModule.js`, `my-component.js` |
| C# | PascalCase | `MyClass.cs` |
| Java | PascalCase (class) | `MyClass.java` |
