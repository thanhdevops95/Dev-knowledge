## 📖 Thuật ngữ

| Từ viết tắt | Tiếng Anh đầy đủ | Nghĩa tiếng Việt |
|-------------|------------------|------------------|
| **Observability** | Observability | Khả năng quan sát hệ thống |
| **Metrics** | Metrics | Chỉ số (CPU, RAM, requests) |
| **Logs** | Logs | Nhật ký sự kiện |
| **Traces** | Distributed Traces | Theo dõi request qua nhiều services |
| **Prometheus** | Prometheus | Hệ thống monitoring metrics |
| **Grafana** | Grafana | Dashboard visualization |
| **ELK** | Elasticsearch + Logstash + Kibana | Stack cho centralized logs |
| **Loki** | Grafana Loki | Log aggregation (lightweight) |
