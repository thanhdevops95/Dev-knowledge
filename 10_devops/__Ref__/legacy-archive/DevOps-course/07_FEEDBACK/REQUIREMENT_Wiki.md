# 📋 MODULE 07: FEEDBACK - YÊU CẦU & TIÊU CHÍ NGHIỆM THU

## 🎯 Mục tiêu Module

1. ✅ Hiểu feedback loops trong DevOps
2. ✅ Implement ChatOps (Slack/Teams integration)
3. ✅ Conduct effective incident post-mortems
4. ✅ Continuous improvement với retrospectives
5. ✅ Measure DORA metrics (Deployment Frequency, Lead Time, MTTR, Change Failure Rate)

## 📖 Thuật ngữ

| Từ viết tắt | Tiếng Anh đầy đủ | Nghĩa tiếng Việt |
|-------------|------------------|------------------|
| **ChatOps** | ChatOps | Vận hành qua chat (Slack, Teams) |
| **Post-Mortem** | Post-Mortem | Họp rút kinh nghiệm sau sự cố |
| **RCA** | Root Cause Analysis | Phân tích nguyên nhân gốc rễ |
| **Blameless** | Blameless Culture | Văn hóa không đổ lỗi |
| **MTTR** | Mean Time To Recovery | Thời gian trung bình phục hồi |
| **DORA** | DevOps Research & Assessment | Chỉ số đánh giá DevOps |
| **SLO** | Service Level Objective | Mục tiêu chất lượng dịch vụ |
| **Error Budget** | Error Budget | Ngân sách lỗi cho phép |
