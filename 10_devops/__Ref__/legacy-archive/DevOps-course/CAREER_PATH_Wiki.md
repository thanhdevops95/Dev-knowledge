## 🎯 Lộ trình 3-5 năm

### Year 1: Foundation & Practice (Junior)

**Focus: Làm chủ công cụ cơ bản**

- ✅ Hoàn thành khóa học này
- ✅ Build 3-5 personal projects
- ✅ Contribute to open source (GitHub)
- ✅ Get first DevOps job (intern/junior)

**Skills to master:**

- Git workflow (PR reviews, merge strategies)
- Docker & Docker Compose thành thạo
- Linux CLI & Bash scripting
- Basic CI/CD (GitHub Actions/GitLab CI)
- AWS fundamentals (EC2, S3, RDS)

**Certifications:**

- [ ] AWS Certified Cloud Practitioner
- [ ] Docker Certified Associate (DCA)

---

### Year 2-3: Depth & Breadth (Mid-level)

**Focus: Kubernetes + Automation + Multi-cloud**

**Skills to add:**

- Kubernetes production experience
- Terraform advanced patterns
- Configuration management (Ansible/Chef)
- Monitoring at scale (Prometheus/Datadog)
- GitOps (ArgoCD/Flux)
- Security basics (HTTPS, secrets management)

**Certifications:**

- [ ] Certified Kubernetes Administrator (CKA)
- [ ] AWS Certified Solutions Architect Associate
- [ ] HashiCorp Certified: Terraform Associate

**Projects:**

- Deploy multi-tier app to K8s
- Build reusable Terraform modules
- Set up complete observability stack

---

### Year 4-5: Specialization (Senior)

**Focus: Architecture + Leadership + Business impact**

**Pick 1-2 specializations:**

#### Option A: Platform Engineering

- Build internal developer platforms
- Service mesh (Istio/Linkerd)
- Developer experience tools

#### Option B: SRE (Site Reliability Engineering)

- SLIs/SLOs/Error budgets
- Chaos engineering
- Performance optimization

#### Option C: Security (DevSecOps)

- CI/CD security scanning
- Compliance automation
- Zero-trust architecture

**Certifications:**

- [ ] Certified Kubernetes Security Specialist (CKS)
- [ ] AWS Certified DevOps Engineer Professional
- [ ] CKA & CKAD if going SRE

**Leadership skills:**

- Mentor junior engineers
- Tech talks & documentation
- Cross-team collaboration

## 💼 Job Search Strategy

### Building Your Portfolio

**Must-have projects:**

1. **Personal website** (your.name.com)
   - Deploy via CI/CD pipeline
   - Show off your infra knowledge

2. **Microservices project**
   - 3-4 services
   - K8s deployment
   - Full monitoring

3. **Open source contributions**
   - Fix bugs in popular tools
   - Write documentation
   - Build integrations

### Resume Tips

**Good DevOps resume includes:**

- ✅ Metrics: "Reduced deploy time from 2h → 15min"
- ✅ Tech stack: "Terraform, K8s, AWS, GitLab CI"
- ✅ Impact: "Enabled 20 deployments/day (from 2/week)"
- ✅ Links: GitHub, LinkedIn, portfolio site

**Avoid:**

- ❌ "Worked with DevOps tools"
- ❌ No quantifiable results
- ❌ Generic descriptions

## ❓ FAQ

**Q: Cần bằng IT không?**
A: Không bắt buộc, nhưng có thì dễ hơn. Nhiều DevOps tự học.

**Q: Cần biết lập trình không?**
A: Cần hiểu code cơ bản (Python/Bash). Nhưng không cần giỏi như Dev.

**Q: DevOps vs SRE khác gì?**
A: SRE chú trọng reliability, metrics, on-call. DevOps rộng hơn, bao gồm CI/CD, IaC.

**Q: Nên bắt đầu từ đâu nếu đang là Dev?**
A: Học Docker → K8s → CI/CD. Leverage coding skills.

**Q: Nên bắt đầu từ đâu nếu đang là SysAdmin?**
A: Học automation (Ansible/Terraform) → Docker → CI/CD.
