# ❓ Quiz: System Design

---

### Q1

99.9% availability nghĩa là downtime mỗi năm?

- a) 5 minutes
- b) 52 minutes
- c) 8.76 hours
- d) 3.65 days

### Q2

RTO là gì?

- a) Recovery Point Objective (Mục tiêu điểm khôi phục - dữ liệu)
- b) Recovery Time Objective (Mục tiêu thời gian khôi phục)
- c) Real Time Operations
- d) Remote Target Objective  

### Q3

Horizontal scaling là?

- a) Bigger instance (Instance lớn hơn)
- b) More instances (Nhiều instance hơn)
- c) Better network (Mạng tốt hơn)
- d) More storage (Nhiều lưu trữ hơn)  

### Q4

Active-Active DR có đặc điểm gì?

- a) One region active (Một vùng hoạt động)
- b) Both regions handle traffic (Cả hai vùng xử lý traffic)
- c) Cold standby
- d) Backup only  

### Q5

SLI là gì?

- a) Service Level Indicator (Chỉ số mức dịch vụ - đo lường thực tế)
- b) Service Level Interface
- c) System Level Integration
- d) Service Log Index
