# 💪 Exercises: Security in Pipeline

---

## Exercise 1: Secret Scanning ⭐

Setup secret scanning (Thiết lập quét secret):

- Add Gitleaks to CI (Thêm Gitleaks vào CI)
- Create pre-commit hook (Tạo pre-commit hook)
- Test with dummy secrets (Test với secrets giả)

## Exercise 2: SAST Integration ⭐⭐

Integrate SAST tools (Tích hợp công cụ SAST):

- SonarQube analysis (Phân tích SonarQube)
- GitHub CodeQL
- Block PR on critical issues (Chặn PR nếu có lỗi nghiêm trọng)

## Exercise 3: Dependency Scanning ⭐⭐

Implement dependency checks (Thực hiện kiểm tra dependency):

- npm audit / pip audit
- Snyk integration (Tích hợp Snyk)
- Automated PRs for updates (Tự động tạo PR để cập nhật)

## Exercise 4: Container Scanning ⭐⭐⭐

Secure container images (Bảo mật container images):

- Trivy in CI pipeline (Trivy trong CI pipeline)
- Block on CRITICAL vulnerabilities (Chặn các lỗ hổng CRITICAL)
- Generate SBOM (Tạo SBOM)

## Exercise 5: Complete DevSecOps Pipeline ⭐⭐⭐

Build full pipeline (Xây dựng pipeline đầy đủ):

- All security scans integrated (Tích hợp tất cả các loại quét bảo mật)
- Quality gates (Cổng chất lượng)
- Security reports (Báo cáo bảo mật)
- Notifications (Thông báo)
