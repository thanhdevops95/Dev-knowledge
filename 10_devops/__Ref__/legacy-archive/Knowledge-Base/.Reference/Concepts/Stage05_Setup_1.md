# 🛠️ GIAI ĐOẠN 5: CHUẨN BỊ FRONTEND

## 📌 MỤC TIÊU
Trong giai đoạn này, chúng ta sẽ thêm giao diện người dùng. Bạn cần tạo các thư mục để chứa code HTML/CSS/JS.

---

## 1. TẠO CẤU TRÚC THƯ MỤC

Tại thư mục gốc dự án:

```bash
# Tạo thư mục frontend
mkdir -p frontend/css
mkdir -p frontend/js
mkdir -p frontend/images (Optional)

# Tạo thư mục config NGINX
mkdir -p nginx/conf.d
```

## 2. KIỂM TRA
Cấu trúc cây thư mục nên như sau:

```text
Todo-App-DevOps/
├── frontend/
│   ├── css/
│   ├── js/
│   └── images/
├── nginx/
│   └── conf.d/
├── go-service/
├── python-service/
├── data/
└── docker-compose.yaml
```

Sẵn sàng để biến ứng dụng trở nên đẹp đẽ!
