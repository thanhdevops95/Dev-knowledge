# Module 15: SRE Scenarios

---

## 🚨 Scenario 1: "Error budget exhausted"

### 💡 Actions

1. **Freeze deployments** until budget recovers
2. Focus on **reliability work**:
   - Fix known issues
   - Add more tests
   - Improve monitoring
3. Review recent changes

---

## 🚨 Scenario 2: "Incident without runbook"

### 💡 Response

1. Document as you go
2. After resolution, create runbook
3. Add to on-call training

**Template:**

```markdown
## Alert: [Name]
## Symptoms
## Investigation Steps
## Resolution
## Escalation
```

---

## 🚨 Scenario 3: "Same incident recurring"

### 💡 Root Cause Analysis

1. Review past postmortems
2. Check if action items completed
3. Dig deeper into root cause
4. Consider architectural changes

---

## 📋 SRE Troubleshooting

| Issue | Action |
|-------|--------|
| Budget used | Pause deploys |
| No runbook | Create one |
| Recurring issue | Better root cause |
