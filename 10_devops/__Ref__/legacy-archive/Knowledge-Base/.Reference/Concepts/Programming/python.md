## Giới thiệu

### Python là gì?

Python là ngôn ngữ lập trình high-level, interpreted, và general-purpose được tạo bởi Guido van Rossum năm 1991.

**Đặc điểm:**
- **Easy to learn**: Cú pháp đơn giản, dễ đọc
- **Versatile**: Web, automation, data science, AI/ML
- **Rich ecosystem**: Hàng nghìn libraries
- **Cross-platform**: Windows, Linux, macOS

### Tại sao DevOps cần Python?

1. **Automation**: Tự động hóa tasks
2. **Scripting**: Viết scripts nhanh
3. **APIs**: Tương tác với REST APIs
4. **Infrastructure**: Boto3 (AWS), Google Cloud Client
5. **CI/CD**: Jenkins, GitLab CI scripts
