# Quiz: Module 03 - NETWORKING INTRO

> **20 Câu Trắc Nghiệm Networking**

**Thời gian:** 20 phút | **Đạt:** 14/20

---

1. Port mặc định cho HTTP?
   - **C) 80**

2. DNS service làm gì?
   - **B) Chuyển domain thành IP**

3. 127.0.0.1 còn gọi là?
   - **C) localhost**

4. Lệnh nào test connectivity?
   - **A) ping**

5. Check open ports?
   - **B) netstat -tulpn**

6. UFW là gì?
   - **A) Firewall**

7. Port HTTPS?
   - **B) 443**

8. DNS lookup command?
   - **C) nslookup**

9. Private IP range?
   - **A) 192.168.x.x**

10. HTTP request từ command line?
    - **B) curl**

11. traceroute hiển thị gì?
    - **C) Network path**

12. Trước enable UFW, phải?
    - **A) Allow port 22**

13. dig dùng để?
    - **B) DNS lookup**

14. lsof -i :80 làm gì?
    - **C) Hiển thị process dùng port 80**

15. 8.8.8.8 là gì?
    - **A) Google DNS**

16. systemctl manages?
    - **B) Services**

17. 502 Bad Gateway nghĩa?
    - **C) Backend không respond**

18. SSL certificate tool?
    - **A) certbot**

19. Load balancer làm gì?
    - **B) Phân phối traffic**

20. DNS config file?
    - **C) /etc/resolv.conf**
