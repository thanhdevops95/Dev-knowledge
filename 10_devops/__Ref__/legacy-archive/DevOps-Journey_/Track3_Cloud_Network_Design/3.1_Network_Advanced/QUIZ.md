### Q1: VPC Definition

What is VPC?

*(VPC là gì?)*

### Q2: Security Group Stateful

Security Group is stateful, what does this mean?

*(Security Group là stateful, có nghĩa gì?)*

- a) Rules are ordered *(Các quy tắc có thứ tự)*
- b) Return traffic automatically allowed *(Lưu lượng trả về tự động được phép)*
- c) Only deny rules *(Chỉ có các quy tắc chặn)*
- d) Subnet level *(Cấp độ Subnet)*

### Q3: NAT Gateway Purpose

What is NAT Gateway used for?

*(NAT Gateway dùng để?)*

- a) Block traffic *(Chặn lưu lượng)*
- b) Allow private instances to access internet *(Cho phép instance private truy cập internet)*
- c) Load balance *(Cân bằng tải)*
- d) VPN connection *(Kết nối VPN)*

---

### Q4: Layer 7 Load Balancer

What can a Layer 7 load balancer route based on?

*(Layer 7 load balancer có thể route dựa trên?)*

- a) IP only *(Chỉ IP)*
- b) Port only *(Chỉ Port)*
- c) URL path, headers *(Đường dẫn URL, headers)*
- d) MAC address *(Địa chỉ MAC)*
