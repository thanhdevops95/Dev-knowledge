# 💪 Exercises: Portfolio Launch

> **Practice Exercises for Building Your Portfolio**
>
> *Bài tập thực hành xây dựng Portfolio.*

---

## Exercise 1: GitHub Profile README (Beginner)

Create a professional GitHub profile README with:

*(Tạo GitHub profile README chuyên nghiệp với:)*

1. Professional introduction *(Giới thiệu chuyên nghiệp)*
2. Skills badges *(Badges kỹ năng)*
3. GitHub stats *(Thống kê GitHub)*
4. Featured projects *(Dự án nổi bật)*

---

## Exercise 2: Portfolio Project (Beginner)

Create a DevOps portfolio project:

*(Tạo dự án portfolio DevOps:)*

1. Pick a topic (CI/CD, IaC, etc.) *(Chọn chủ đề)*
2. Write detailed README *(Viết README chi tiết)*
3. Include architecture diagram *(Thêm sơ đồ kiến trúc)*
4. Add deployment instructions *(Thêm hướng dẫn triển khai)*

---

## Exercise 3: LinkedIn Optimization (Beginner)

Optimize your LinkedIn profile:

*(Tối ưu hóa profile LinkedIn:)*

1. Professional headline *(Tiêu đề chuyên nghiệp)*
2. Summary with keywords *(Tóm tắt với từ khóa)*
3. Add skills and certifications *(Thêm kỹ năng và chứng chỉ)*
4. Request recommendations *(Yêu cầu lời giới thiệu)*

---

## Exercise 4: Technical Blog Post (Intermediate)

Write a technical blog post about:

*(Viết bài blog kỹ thuật về:)*

1. A problem you solved *(Vấn đề bạn đã giải quyết)*
2. Step-by-step tutorial *(Hướng dẫn từng bước)*
3. Code examples *(Ví dụ code)*
4. Lessons learned *(Bài học rút ra)*

---

## Exercise 5: Open Source Contribution (Intermediate)

Make your first open source contribution:

*(Thực hiện đóng góp open source đầu tiên:)*

1. Find a project *(Tìm một dự án)*
2. Read contribution guidelines *(Đọc hướng dẫn đóng góp)*
3. Fix a small issue *(Sửa một issue nhỏ)*
4. Submit pull request *(Gửi pull request)*

---

## ✅ Submission Checklist

- [ ] GitHub profile README live *(README profile GitHub hoạt động)*
- [ ] 1+ portfolio project complete *(1+ dự án portfolio hoàn thành)*
- [ ] LinkedIn profile updated *(Profile LinkedIn đã cập nhật)*
- [ ] Blog post published *(Bài blog đã xuất bản)*
- [ ] 1 open source contribution *(1 đóng góp open source)*
