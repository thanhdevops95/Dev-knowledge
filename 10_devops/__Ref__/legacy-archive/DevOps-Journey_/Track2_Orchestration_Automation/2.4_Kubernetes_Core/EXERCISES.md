# 💪 Exercises: Kubernetes Core

## Exercise 1: Deploy Application ⭐

Deploy NGINX với 3 replicas và expose via LoadBalancer (Triển khai NGINX với 3 replicas và expose qua LB).

## Exercise 2: ConfigMap & Secrets ⭐⭐

## Exercise 3: Persistent Storage ⭐⭐

## Exercise 4: Multi-tier App ⭐⭐⭐

Deploy frontend + backend + database với proper networking (Triển khai frontend + backend + DB với networking chuẩn).

## Exercise 5: Resource Management ⭐⭐⭐

Set resource limits, liveness/readiness probes (Thiết lập resource limits và probes).
