# EXERCISES - Module 02: GIT & GITHUB

> **Purpose:** Master Git version control through practice
> **Difficulty:** ⭐⭐☆☆☆ to ⭐⭐⭐⭐☆ | **Time:** 3-4 hours

---

## 📋 20 Exercises

### Basic Git Operations (1-5)

**Exercise 1:** Repository Setup

- Initialize Git repo
- Make 5 commits with proper messages
- View commit history with `git log --oneline --graph`

**Exercise 2:** Ignoring Files

- Create Node.js project
- Add appropriate `.gitignore`
- Verify ignored files don't appear in `git status`

**Exercise 3:** Undoing Changes

- Make changes to file
- Discard changes with `git restore`
- Stage file, then unstage with `git restore --staged`

**Exercise 4:** Amending Commits

- Make commit
- Realize you forgot a file
- Use `git commit --amend` to fix

**Exercise 5:** Stashing

- Make changes
- Stash with message
- Do other work
- Pop stash and apply

### Branching & Merging (6-10)

**Exercise 6:** Feature Branch Workflow

- Create `feature/login` branch
- Make 3 commits
- Merge to `main`
- Delete feature branch

**Exercise 7:** Merge Conflict

- Create two branches modifying same file lines
- Merge and resolve conflict manually
- Complete merge

**Exercise 8:** Rebasing

- Create feature branch
- Make commits on main
- Rebase feature onto main
- Resolve any conflicts

**Exercise 9:** Cherry-pick

- Find specific commit in branch
- Cherry-pick to another branch
- Verify changes applied

**Exercise 10:** Branch Management

- List all branches (local + remote)
- Delete merged branches
- Rename branch

### GitHub Collaboration (11-15)

**Exercise 11:** Fork & PR Workflow

- Fork public repository
- Clone your fork
- Make improvements
- Create Pull Request

**Exercise 12:** Code Review

- Review a PR (yours or classmate's)
- Request changes
- Approve after fixes

**Exercise 13:** Syncing Fork

- Add upstream remote
- Fetch upstream changes
- Merge into your fork
- Push to origin

**Exercise 14:** Releases & Tags

- Create annotated tag
- Push tag to GitHub
- Create release with notes

**Exercise 15:** GitHub Issues

- Create issue template
- Open issue
- Link PR to issue
- Close via commit message

### Advanced Git (16-20)

**Exercise 16:** Interactive Rebase

- Make 5 small commits
- Squash into 1 meaningful commit
- Force push to feature branch

**Exercise 17:** Git Bisect

- Introduce bug in commit
- Use git bisect to find it
- Fix bug

**Exercise 18:** Reflog Recovery

- Delete branch
- Use reflog to find commit
- Recover deleted branch

**Exercise 19:** Git Hooks

- Create pre-commit hook that runs linter
- Test hook prevents bad commits
- Create commit-msg hook for message format

**Exercise 20:** Git Aliases

- Create 10 useful aliases
- Add to `.gitconfig`
- Document each alias purpose
