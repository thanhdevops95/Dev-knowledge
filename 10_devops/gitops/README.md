> 🎯 *GitOps là một phương pháp luận vận hành hệ thống, sử dụng kho lưu trữ Git làm nguồn chân lý duy nhất (Single Source of Truth) cho cấu hình hạ tầng và ứng dụng. Nó áp dụng các quy trình phát triển phần mềm (commit, PR, code review) trực tiếp vào hoạt động vận hành hệ thống.*
